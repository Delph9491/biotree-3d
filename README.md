# 🌳 BioTree 3D Prototype

ต้นไม้อัจฉริยะ - ฟอกอากาศด้วยพลังงานแสงอาทิตย์

## 📋 ภาพรวมโปรเจกต์

BioTree เป็นอุปกรณ์ฟอกอากาศอั���ฉริยะที่ออกแบบมาเพื่อดูแลคุณภาพอากาศในบ้าน โครงการนี้ประกอบด้วย Visualization แบบ 3D ที่โต้ตอบได้เพื่อแสดงส่วนประกอบและสถานะของอุปกรณ์

## 🎯 คุณสมบัติหลัก

✅ **แบบจำลอง 3D แบบโต้ตอบ** - สามารถหมุน ขยาย ย่อ ได้
✅ **ระบบฟอกอากาศ** - แสดงไส้กรอง HEPA และคาร์บอน
✅ **แผงโซลาร์เซลล์** - พลังงานแสงอาทิตย์
✅ **หน้าจอแสดง PM2.5** - คุณภาพอากาศแบบเรียลไทม์
✅ **เอฟเฟกตอนุภาค** - อนุภาคออกซิเจนลอยฟุ้ง
✅ **Responsive Design** - ใช้ได้กับ Desktop, Tablet, Mobile

## 📦 ส่วนประกอบของโมเดล 3D

### 1. ฐาน (Base)
- ภาชนะโปร่งใส 45x45 cm
- ประกอบด้วย:
  - ไส้กรอง HEPA (wireframe)
  - ไส้กรอง Carbon (สีเทาเข้ม)
  - แบตเตอรี่

### 2. ลำต้น (Stem)
- ท่อไม้ยาว 4 เมตร
- หน้าจอแสดง PM2.5 (ข้อมูลคุณภาพอากาศ)

### 3. พุ่มใบไม้ (Foliage)
- ทรง Dodecahedron
- ใบไม้ 5 ชิ้น (สีเขียว)
- แผงโซลาร์เซลล์ 3 แผง (สีน้ำเงิน)

### 4. อนุภาค (Particles)
- 150 ละอองออกซิเจน
- แสดงการไหลของอากาศสะอาด
