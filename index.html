ต้นไม้อัจฉริยะ - ฟอกอากาศด้วยพลังงานแสงอาทิตย์

เกี่ยวกับ BioTree

BioTree เป็นอุปกรณ์ฟอกอากาศอัจฉริยะที่ออกแบบมาเพื่อดูแลคุณภาพอากาศในบ้าน โดยใช้เทคโนโลยี 3D Visualization เพื่อแสดงสถานะของอุปกรณ์และคุณภาพอากาศในเวลาจริง

ด้านล่างคุณสามารถดูแบบจำลอง 3D ของต้นไม้และโต้ตอบกับมันได้โดยการเลื่อนและหมุน

⚡ แผงโซลาร์เซลล์

สร้างพลังงานจากแสงอาทิตย์เพื่อให้อุปกรณ์ทำงาน

🌬️ ระบบฟอก

ไส้กรอง HEPA และคาร์บอนเพื่อฟอกอากาศอย่างมีประสิทธิภาพ

📊 หน้าจอแสดงผล

แสดงค่า PM2.5 และสถานะคุณภาพอากาศในเวลาจริง

🎯 ตัวอย่าง 3D Model

วิธีโต้ตอบกับโมเดล:

  • บนคอมพิวเตอร์: ใช้เมาส์เพื่อหมุน, scroll เพื่อขยาย/ย่อ
  • บน iPad/Mobile: ใช้นิ้วเพื่อหมุน, pinch เพื่อขยาย/ย่อ

💻 เทคโนโลยีที่ใช้

โปรเจกต์นี้ใช้ Three.js library เพื่อสร้างและแสดงผล 3D graphics:

// ไลบรารี่ที่ใช้: • Three.js - WebGL library สำหรับ 3D graphics • OrbitControls - ควบคุมการหมุนและการมองดูด้วยเมาส์และการทัช // คุณสมบัติหลัก: ✓ Smooth animations ✓ Interactive 3D visualization ✓ Touch support สำหรับ iPad ✓ Responsive design ✓ Particle effects (oxygen simulation) ✓ Real-time rendering

ส่วนประกอบหลักของโมเดล:

1. ฐาน (Base) - ภาชนะโปร่งใส 45x45 cm ├── ไส้กรอง HEPA (wireframe สีเทา) ├── ไส้กรอง Carbon (สีเทาเข้ม) └── แบตเตอรี่ (สีเทา) 2. ลำต้น (Stem) - ท่อไม้ยาว 4 เมตร └── หน้าจอแสดงผล PM2.5 (สีดำ/เขียว) 3. พุ่มใบไม้ (Foliage) - ทรง Dodecahedron ├── ใบไม้ 5 ชิ้น (สีเขียว) └── แผงโซลาร์เซลล์ (สีน้ำเงิน) 4. อนุภาค (Particles) - ออกซิเจนลอยฟุ้ง └── 150 ละอองแสดงการไหลของอากาศสะอาด
